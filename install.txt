1) instalar as ferramentas
    - mininet
    - ryu
    - mongodb

2) Configurar arquivo de config

3) Configurar ferramenta
    - colocar config em /etc
    - colocar controlador no lugar certo
    - criar pasta de log em /var/log
    - criar uds socket em /var/run
    - linkar executavel



4) configurar o banco de dados
    - criar banco de dados
    - configurar mongodb.conf bind_ip 0.0.0.0 para permitir a comunicação na interface de loopback (já que estamos executando em uma única máquina)
